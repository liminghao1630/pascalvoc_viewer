# pascalvoc_viewer
A viewer of Object Detection pascal voc format data.

# Usage
Change the data_dir into your dataset directory.  
The structure of data_dir should be like following:

data_dir
|-- images
|     |-- example-001.jpg
|     |-- example-002.jpg
|       ...
|
|-- annotations
|     |-- example-001.xml
|     |-- example-002.xml
|       ...
